# crispies

Repo for small web apps and tools.

## Fantasy Basketball

Per-game stats analyzer with configurable league scoring lives in [`fantasy-basketball/`](fantasy-basketball/). Set your points-per-stat and see fantasy PPG for each player.

## HIIT Timer

A mobile-friendly HIIT workout timer lives in [`timer/`](timer/).

### Use it on your phone (web app)

1. **Enable GitHub Pages** for this repo:
   - On GitHub: **Settings → Pages**
   - **Source**: Deploy from a branch
   - **Branch**: `main` (or your default), folder **/ (root)**
   - Save. After a minute or two the site will be live.

2. **Open on your phone**  
   URL: **https://bigcriispy.github.io/crispies/**  
   (Replace `bigcriispy` with your GitHub username if the repo is under a different account.)

3. **Add to home screen** (optional)  
   On iPhone: Safari → Share → **Add to Home Screen**.  
   On Android: Chrome → menu (⋮) → **Add to home screen** or **Install app**.  
   Then open the icon like an app; it runs in standalone mode without the browser UI.
